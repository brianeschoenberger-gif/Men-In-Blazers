# Men In Blazers - Cinematic Homepage

Milestones 0-2 scaffold for a premium soccer media homepage built with:
- React + Vite + TypeScript
- React Three Fiber / Three.js
- GSAP ScrollTrigger

## Run
```powershell
cmd /c npm install
cmd /c npm run dev
```

## Build
```powershell
cmd /c npm run build
```

## Quality Gates
```powershell
cmd /c npm run lint
cmd /c npm run build
cmd /c npm run test:smoke
```

GitHub Actions also runs both checks on every push and pull request.
